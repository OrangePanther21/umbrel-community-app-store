## RoboSats Tools - Umbrel Community App Store

This is an Umbrel Community App Store for unofficial tools in the RoboSats ecosystem. These additional app stores allow developers to distribute applications without submitting to the [Official Umbrel App Store](https://github.com/getumbrel/umbrel-apps).

## App Store Details

- **ID**: `robosats-tools`
- **Name**: RoboSats Tools
- **Tagline**: Unofficial tools for RoboSats ecosystem

## Available Apps

### RoboSats Notifier

Monitor RoboSats order book and receive instant notifications when new offers match your criteria.

- **App ID**: `robosats-tools-robosats-notifier`
- **Features**: 
  - Real-time monitoring of RoboSats offers across multiple coordinators
  - Instant notifications for new matching offers (WhatsApp supported)
  - Web UI for easy configuration
  - QR code authentication
  - Filter by currencies and coordinators
  - Customizable check intervals
